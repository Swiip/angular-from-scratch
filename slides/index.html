Étape 2

$watch et $digest

  • Le framework
    • Stocker les "$$watchers" dans le scope
    • Définir la fonction $watch dans le prototype du scope
    • Définir la fonction $digest dans le prototype du scope
  • Utilisation du framework
    • Surveiller les modifications d'une donnée dans le scope
    • Lancer la digestion

Étape 2

Le framework


Scope.prototype.$watch =
    function(watchFn, listenerFn) {
        this.$$watchers.push({
            watchFn: watchFn,
            listenerFn: listenerFn
        });
    };

Étape 2

Le framework


Scope.prototype.$digest =
    var scope = this;
    function() {
        _.each(this.$$watchers, function(watch){
            var newValue = watch.watchFn(scope);
            if (watch.last !== newValue) {
                watch.listenerFn(newValue,
                      watch.last, scope);
            }
            watch.last = newValue;
        });
    }

Étape 2

Utilisation du framework


myScope.$watch(function() {
    return myScope.firstName;
}, function(newValue, oldValue, scope) {
    console.log('firstName changed from',
        oldValue, 'to', newValue);
});

myScope.$digest();

Étape 3

Digest loop !

  • Le framework
    • Définir si une "digestion" à occasionner une modification
    • Boucler la "digestion" tant qu'il y a des modifications
    • Mettre en place un garde fou contre les boucles infinie
  • Utilisation du framework
    • Réaliser un $watch qui déclenche une nouvelle modification
    • Réaliser deux $watch qui créent une boucle infinie

Étape 2

Le framework


do {
    dirty = false;
    _.each(this.$$watchers, function (watcher) {
        var newValue = watcher.watcherFn(scope);
        if (watcher.last !== newValue) {
            dirty = true;
            watch.listenerFn(newValue,
                  watch.last, scope);
            watcher.last = newValue;
        }
    });
} while (dirty);

Étape 4

Let's get dirty !

  • Le framework
    • Ajouter une propriété byValue dans les watchers
    • Si byValue est true, réaliser un clone de la valeur
    • Si byValue est true, réaliser une comparaison en profondeur (Underscore est votre ami)
  • Utilisation du framework
    • Réaliser une modification dans le scope détectable seulement byValue

Étape 4

Le framework


Scope.prototype.$watch =
    function(watchFn, listenerFn, byValue) {
        this.$$watchers.push({
            watchFn: watchFn,
            listenerFn: listenerFn,
            byValue: byValue
        });
    };
